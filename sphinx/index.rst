Welcome to quickdb's documentation!
===================================

QuickDB is an implementation of distributed database
of map-reduce architecture in Python.
With this system, users can give mapper and reducer
in Python code directly to the system to process data.
It also has an interface of SQL.

Codes are available at `GitHub <https://github.com/michitaro/quickdb/>`_.
